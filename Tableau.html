TABLEAU

Un ensemble des lignes qui contient des cellules ( les colomnes n'existent pas).Pour chaque element du tableau nous avons une balise:

			< tableau >
				< tr >
					< td >  Nom </td>
					< td >  Prenom </td>
				< /tr >
			< /tableau >

Exemple:

tableau < tableau >
ligne < tr >
cellule < td >

Ligne d'entête et ligne de pied

Pour creer un entête nous devons faire la commande suivante:

			< tableau >
				< thead >
					< th >  Nom </th>
					< th >  Prenom </th>
				< /thead >
			< /tableau >

Pour creer une ligne de pied nous devons faire la commande suivante:

			< tableau >	
				< tfoot >
					< th > </th>
					< th > </th>
				< /foot >
			< /tableau >

Pas importe ou se deplace "thead" ou "tfoot" ils vont se mettre dans la même position par defaut. Par contre il va toujours être lu en premier, et il doit être ècrit dans la premier ligne après la declaration de la balise tableau.

Parfois nous avons besoin de surligner une ligne d'abord, pour cela nous devons utiliser la commande suivante:

			< tableau >	
				< tr >
					< th > A </th>
					< td > </td>
				< /tr >
			< /tableau >

Fusionner des cellules

Nous utilisons des attributs pour savoir la postion de largeur et longeur:

			< td rowspan = "2"
					colspan = "2" / >

Exemple:

tableau < tableau >
ligne < tr >
cellule < td >

Ajouter une legende dans le tableau

Dans la balise table nous ecrivons une balise caption avec le texte de la legende. Nous devons l'ecrire seulement au debut de la declaration de la table et il faut que soit un enfent direct de notre table

			
				< caption > Ma legende </caption >
